本とスマホで資格勉強を効率化！

インターネット上に問題集のコンテンツがなくても、 お手持ちの「本」の情報を読み込むだけで、自分だけの電子問題集が作れます。

正誤の履歴が残るので、苦手な部分を中心に効率的に資格勉強を行うことができます。

必要なのは「答え」だけ！

問題データはCSVデータに入力して読み込みます。 その際に使用するデータには「問題文」は必須ではありません。

「答え」の情報だけ登録しておけば、問題文と解説を本の問題集・参考書を見ながら、 問題を解き、結果の記録と答え合わせを効率的に行うことが出来ます。
